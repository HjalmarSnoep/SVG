WAT IS SVG?

S.V.G. staat voor Scaleable Vector Graphics. Vector, omdat deze tegenover Raster plaatjes (of graphics of bitmaps, JPG,PNG, GIF)) staan.

Op het web van vandaag is SVG steeds belangrijker, om de volgende redenen:

  • Responsive design

    SVG is standaard responsive en biedt vele mogelijkheden die HTML niet of niet goed biedt.

  • HD web

    Op HD schermen wil je scherpe plaatjes, op telefoons met kleine schermen ook. RasterImages of bitmaps (JPG,PNG, GIF) bieden hier geen goede oplossing voor. Bij SVG is schalen ingebouwd.

  • Programmeren en interactiviteit

    SVG maakt standaard interactie met grillige vormen mogelijk. Ideaal voor mensen die -out-of-the-box- willen denken.

  • SEO, Leesbaarheid

    Text in een SVG is WEL leesbaar door google. Maak je een button met SVG, dan weet google dus wat er op staat.

  • Klein

    Als je geen foto wilt, is een SVG eigenlijk altijd kleiner dan een bitmap. Erg cool voor achtergronden. Ook is het text en de meeste servers zenden tekst tegenwoordig gecomprimeerd over. Een snellere site komt hoger in google!

  • Icons

    Font Awesome is mooi, maar alleen als iemand het al geladen heeft, anders haal je voor dat ene facebook-icoontje een heel font binnen. Ook zitten niet ALLE iconen in fontAwesome. Bovendien biedt SVG spriting, hetgeen ervoor zorgt, dat je iconen ook buiten de SVG kan gebruiken.

  • DE standaard

    SVG wordt tegenwoordig door ALLE grote browsers erg goed ondersteund en wordt gebruikt als fallback voor webfonts. Ook is het hele fundament ERG GOED ONDERSTEUND

  • Speciale effecten

    SVG kan gebruikt worden om speciale effecten te maken, die niet (goed) in HTML mogelijk zijn.

  • Luistert naar CSS

    SVG eigenschappen kunnen worden overschreven door CSS. Zo kun je hover-effecten bereiken in een plaatje of verschillende kleuren voor game-assets gebruiken of buttons verkleuren om bij een bepaald thema te passen.

  • Gratis editors

    Er zijn vele gratis editors, waarmee je zelf SVG's kunt tekenen.
