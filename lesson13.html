Text

SVG heeft vele mogelijkheden voor tekst, die je niet zou verwachten.

textpath

Je hebt de mogelijkheden om text op een pad te zetten, zelfs als dat pad animeert!
Dit is in bovenstaand voorbeeld niet heel duidelijk, omdat het gebruik maakt van SVG.js, maar... <textpath> is een standaard SVG attribute, dat je kunt nesten in de <text> tag.
https://developer.mozilla.org/en-US/docs/Web/SVG/Element/textPath

tspan

Je kunt gebieden in text style, zoals je gewent bent in CSS met tspan. Zo kun je woorden binnen een text bold, italic maken en verkleuren enzo.

textarea

Je kunt een tekst area voor meerdere regels tekst en automatisch omloop, zoals je van HTML gewend bent. Als je bang bent dat het niet wordt gesupport (want het is vrij nieuw). Met <g requiredFeatures=""> kun je aangeven welke nieuwe features je gebruikt, zodat je browser een nette melding kan geven als het nog niet wordt ondersteund. In deze tutorial ga ik alleen uit van wat NU goed ondersteund wordt. Dit is dus een uitzondering :)

Andere attributen

Dit zijn attributen die moeilijk uit te leggen zijn, zonder beelden. Klik daarom op de onderstaande voorbeelden.
  • Dominant-baseline & text-anchor
  • letter-spacing
  • text-decoration
  • text-rendering (zelfde als css text-rendering)
  • textLength && lengthAdjust
  • word-spacing
